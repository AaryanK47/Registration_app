# Registration App

## Description
A simple web application for student registration. This application allows students to submit their details including name, email, course, and city.

## Features
- **Student Application Form**: A user-friendly form to collect student information.
- **Form Validation**: Client-side validation using jQuery to ensure all required fields are filled.
- **Submission Handling**: PHP backend script to process and display the submitted data.
- **Responsive Design**: Basic responsive layout for better usability.

## Technologies Used
- HTML5
- CSS3
- JavaScript (jQuery)
- PHP

## How to Run
1.  Ensure you have a local server environment set up (e.g., XAMPP, WAMP, or MAMP) that supports PHP.
2.  Clone or download this repository to your server's root directory (e.g., `htdocs` for XAMPP).
3.  Start your local server (Apache).
4.  Open your web browser and navigate to `http://localhost/registration_app/index.html` (adjust the path based on your folder structure).
5.  Fill out the form and click "Submit Application" to see the result.

## File Structure
- `index.html`: The main registration form.
- `style.css`: Stylesheet for the application.
- `script.js`: JavaScript file for form validation.
- `submit.php`: PHP script to handle form submission and display results.
